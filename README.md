# 🎮 Tetris v4

간단하고 재미있는 **테트리스 게임** 구현 프로젝트입니다.  
HTML5 Canvas와 JavaScript만으로 제작되었으며, 점수와 줄 개수 표시, 화면 흔들림 효과 등 기본적인 테트리스 기능을 제공합니다.

---

## 📌 주요 기능
- **블록 종류**: T, O, L, J, I, S, Z
- **점수 시스템**: 줄을 지울 때마다 점수와 줄 개수 증가
- **화면 흔들림 효과**: 줄 제거 시 시각적 피드백
- **게임 오버 처리**: 블록이 쌓여 충돌 시 게임 초기화
- **키보드 조작**
  - ⬅️ `ArrowLeft`: 왼쪽 이동
  - ➡️ `ArrowRight`: 오른쪽 이동
  - ⬇️ `ArrowDown`: 빠른 하강
  - ⬆️ `ArrowUp`: 회전

---

## 🛠️ 기술 스택
- **HTML5 Canvas**: 게임 화면 렌더링
- **JavaScript (ES6)**: 게임 로직 및 이벤트 처리
- **CSS**: 기본 레이아웃 및 스타일링
